``` r
accel =
  read_csv(
        "./data/accel_data.csv"
        ) %>%
  janitor::clean_names() %>%
  pivot_longer(
    activity_1:activity_1440,
    names_to = "activity_ct",
    values_to = "minute") %>%
  mutate(
    day_num = recode(day, "Sunday" = "7", "Saturday" = "6", "Friday" = "5", "Monday" = "1", "Tuesday" = "2", "Wednesday" = "3", "Thursday" = "4")) %>%
  arrange(week, day_num) %>%
  select(-day_num) %>%
  relocate(minute, .before = "activity_ct") %>%
  mutate(
    weekday_vs_weekend = case_when(
    day %in% c("Monday","Tuesday","Wednesday","Thursday","Friday") ~ "weekday",
    day %in% c("Saturday","Sunday") ~ "weekend")) %>%
  mutate(
    day = as.factor(day))
```

This data collected five weeks of accelerometer data on a 63 year-old
male with BMI 25. After tidy the original data, the resulting dataset
has 50400 observations and 6 variables. Day of the week is the only
factor class variable. Other then `day`, variables `week`, `day_id` and
`minute` are class `numeric`, and variables `activity_ct` and
`weekday_vs_weekend` are class `numeric`. Variable `minute` collected
1440 minutes of each day in the `day` variable over five weeks.

  - Aggregate accross minutes to create a total activity variable for
    each day, and create a table showing these totals.

<!-- end list -->

``` r
total_active =
  accel %>%
    group_by(week, day) %>%
    count(sum(minute)) %>%
  mutate(
    day = as.character(day),
    day_num = recode(day, "Sunday" = "7", "Saturday" = "6", "Friday" = "5", "Monday" = "1", "Tuesday" = "2", "Wednesday" = "3", "Thursday" = "4")) %>%
  arrange(week, day_num) %>%
  select(-day_num, -n) %>%
  janitor::clean_names() %>%
  pivot_wider(names_from = day, values_from = sum_minute)

knitr::kable(total_active)
```
